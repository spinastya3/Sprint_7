# Sprint_7
**Учебный проект 7 спринта для Яндекс.Практикум (API)**

Тестирование API сервиса Яндекс.Самокат https://qa-scooter.praktikum-services.ru/


| **Технология**              | **Версия** |
|-----------------------------|------------|
| Java                        | 11         |
| JUnit                       | 4.13.2     |
| Maven                       | 3.9.12     |
| RestAssured                 | 5.3.0      |
| Allure                      | 2.21.0     |

**Особенности проекта**
* Использован паттерн **Fluent Interface** для моделей данных.
* Реализована автоматическая очистка данных (удаление курьеров и отмена заказов) после тестов.
* Тесты разделены на логические блоки (создание, авторизация, удаление курьера и работа с заказами).

**Запуск**

Чтобы запустить тесты, выполните следующую команду:
`mvn clean test`

**Отчетность**

Чтобы сгенерировать и открыть Allure-отчет, выполните команду:
`allure serve target/allure-results`
